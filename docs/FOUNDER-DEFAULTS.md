# Founder defaults (Ingwaz v0.2)

Зафиксированные продуктовые defaults без опоры на личный опыт founder — для монетизации и фокуса. Источник: ICP CommentSignal, `ROOT-DOMAIN-DECISION.md`, разбор ChatGPT (май 2026).

| # | Вопрос | Default |
|---|--------|---------|
| 1 | Фраза на 10 секунд | Повторяющиеся вопросы в комментариях → готовые темы для следующего видео |
| 2 | Aha moment | Ранжированные **идеи для контента** из кластеров; вторично — вопросы и возражения |
| 3 | Не для нас | Shorts/viral, faceless AI-конвейер, lifestyle без экспертизы, agency workflows |
| 4 | Образ бренда | **Creator research tool** + indie software lab (не «AI platform») |
| 5 | Secondary на главной | Оставить **слабым** списком; не убирать, не делать равными CTA |
| 6 | Evidence | Mock-пример insight до появления прод-скрина CommentSignal (пометка «пример») |
| 7 | Метрика корня | Qualified клики → signup/activation CommentSignal, не vanity impressions |

Пересмотр: [ROOT-DOMAIN-DECISION.md](./ROOT-DOMAIN-DECISION.md) §6.
