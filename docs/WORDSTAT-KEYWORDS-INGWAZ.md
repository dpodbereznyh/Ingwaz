## Выводы (кратко)

1. **Точные product-запросы** («анализ комментариев youtube», «идеи из комментариев») — **0–14 показов/мес**. SEO на формулировку продукта почти не существует; это нормально для новой ниши.
2. **Спрос на «идеи для YouTube»** есть, но **широкий и «мусорный»**: seed «идеи для видео» = **20 840** (гача, TikTok, школьники). Для CommentSignal релевантнее **«идеи для видео на ютуб» (253)**, **«идеи для youtube» (201)**, **«темы для видео youtube» (100)**.
3. **«комментарии ютуб» (17 617)** — intent **зрителя** (как включить, удалить, перевести), **не автора**. Не таргетировать CommentSignal на этот кластер.
4. **«исследование аудитории» (5 132)** и **«обратная связь аудитории» (1 152)** — маркетинг/учёба; для **корня ingwaz** — только естественно в copy, не primary SEO.
5. **«сигнал аудитории» (3 230)** — омоним (ТВ/радио/техника), **не бренд**. В SEO title не опираться.

**Стратегия без изменений:** корень = витрина + UTM; **CommentSignal** = SEO под «идеи/темы для ютуб» + дифференциатор «из комментариев» в title/H1/FAQ, без keyword stuffing.
