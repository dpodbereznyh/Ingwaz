# Именование и позиционирование (корень `ingwaz.space`)

**Статус:** рабочее решение (май 2026). Пересмотр при запуске главной в проде.

## Уровни бренда

| Уровень | Имя | Где |
|---------|-----|-----|
| Домен | `ingwaz.space` | URL, мелко в шапке |
| Витрина / проект (этот репо) | **Сигнал аудитории** | H1, title, коммуникация |
| Продукт-якорь | **CommentSignal** | Главный CTA → `commentsignal.ingwaz.space` |
| Остальные tools | Собственные имена | Блок «Другие инструменты», без ребрендинга |

## Thesis (одна фраза)

> Инструменты **audience intelligence** для авторов экспертного контента — от feedback аудитории к решениям по контенту.

## Черновик для главной (SEO)

| Поле | Черновик (RU) |
|------|----------------|
| **Title** | Сигнал аудитории — инструменты для экспертных авторов \| ingwaz.space |
| **Description** | CommentSignal и смежные утилиты: инсайты из комментариев и feedback, идеи для контента. Для YouTube и экспертных каналов. |
| **H1** | Сигнал аудитории |
| **Подзаголовок** | От комментариев и обратной связи — к решениям по контенту |

## Не использовать

- H1 «Ingawaz» без пояснения смысла
- «Всё для блогеров» / «9 типов creators»
- «Портфель SaaS» как публичная формулировка

## Альтернативы (отложены)

- **Лаборатория аудитории** — шире, слабее связь с CommentSignal
- **Голос аудитории** — понятно, но без линии Signal
- **Ingawaz** как единственный бренд — слабый SEO и смысл
