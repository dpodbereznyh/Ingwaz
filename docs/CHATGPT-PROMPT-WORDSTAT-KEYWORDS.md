# Промпт для ChatGPT: варианты поисковых запросов (Wordstat)

Скопируйте блок **«Задача»** в ChatGPT. Результат сверяйте с Wordstat — см. [WORDSTAT-KEYWORDS-INGWAZ.md](./WORDSTAT-KEYWORDS-INGWAZ.md).

---

## Контекст (можно сократить)

Мы строим:

- **`ingwaz.space`** — минимальная RU-витрина «Сигнал аудитории»; главный CTA → CommentSignal.
- **`commentsignal.ingwaz.space`** — продукт: идеи для YouTube-роликов из комментариев.

**ICP:** авторы **экспертного** контента (образование, B2B, ниша). **Не:** mass-entertainment, viral Shorts.

**Поисковик:** Яндекс, регион Россия.

---

## Задача

Сгенерируй **40 поисковых запросов на русском**, которые могут вводить в Яндекс наш ICP, если ищут:

1. идеи / темы для YouTube-видео;
2. понимание аудитории, обратную связь, комментарии;
3. инструменты для авторов каналов (не монтаж и не «как набрать подписчиков»).

**Формат ответа — таблица Markdown:**

| Запрос | Кластер | Intent (info / tool / commercial) | Для какого URL (ingwaz / commentsignal / не SEO) | Почему релевантно |
|--------|---------|-----------------------------------|--------------------------------------------------|-------------------|

**Кластеры (минимум по 8 запросов в каждом):**

- `youtube_ideas` — идеи и темы для роликов
- `comments_feedback` — комментарии, вопросы аудитории, обратная связь
- `audience_research` — исследование аудитории, «что спрашивают»
- `tools_workflow` — сервисы/утилиты для авторов (без «купить подписчиков»)
- `expert_b2b` — экспертный / образовательный / B2B контент

**Правила:**

- Только **русский**; допустимы «youtube», «ютуб» латиницей/кириллицей как люди реально ищут.
- Без бренда CommentSignal в запросах (мы проверяем demand, не brand).
- Отдельно пометь **5 long-tail** (4+ слова) — их приоритетнее для нишевого продукта.
- Отдельно пометь **5 запросов, которых избегать** (слишком широкие или не наш ICP) — с причиной.
- Не выдумывай частотность Wordstat — только гипотезы формулировок.

**После таблицы:** топ-10 запросов для **первой проверки в Wordstat** (обоснуй в 2–3 предложениях).

---

## После ответа ChatGPT

1. Перенести топ-10 в Wordstat, записать показы в [WORDSTAT-KEYWORDS-INGWAZ.md](./WORDSTAT-KEYWORDS-INGWAZ.md).
2. Если показы нулевые — взять **похожие** из Wordstat по seed «идеи для youtube» / «комментарии ютуб».
3. Победители → FAQ / meta CommentSignal (не keyword stuffing на корне).
